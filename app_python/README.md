# Moscow Time App

## Overview

Python Web App built using Flask Framework and ApiNinjas for requesting Moscow Time

## Usage

1. Install Python 3.9
2. Install pip
3. Install Flask
4. Enter app_python repository
5. Run `flask run`
6. An application will open at http://127.0.0.1:5000/
